멀티모달 생성형 인공지능에서
사운드 기반 생성의 문화적 편향 분석

Cultural Bias Analysis in Sound-Based Generation of Multimodal Generative AI

Paper Code

Abstract

본 연구는 멀티모달 생성형 인공지능에서 사운드 기반 생성에서 발생하는 문화적 편향을 분석한다. 특히 sound-to-image 및 text-to-sound 생성 태스크를 대상으로, 서양권 데이터로 학습된 모델이 비서구 문화적 맥락을 어떻게 반영하는지를 실험적으로 평가한다. 한국 문화 요소가 포함된 입력을 구성하고, 생성 결과를 품질과 의도 정합성 관점에서 분석한 결과, 모델이 문화적 특성을 충분히 반영하지 못하고 서구 중심적 표현으로 치환하는 경향을 확인하였다. 본 연구는 사운드가 포함된 멀티모달 생성 환경에서의 문화적 편향 문제를 정리하고, 향후 공정한 멀티모달 생성 모델 설계를 위한 기초 분석을 제공한다.

Key Findings

  • 품질 (Quality): 모든 모델이 Traditional 범주에서 Contemporary 범주보다 높은 거릿값을 보여 음향/시각적 품질의 불안정성이 관찰됨
  • 정합성 (Alignment): 서구적이고 현대적인 프롬프트일수록 실제 이미지/음향과 더 높은 유사도를 보임
  • 문화적 편향: 한국 전통 소리에 대해 모델들이 문화적 특성을 충분히 반영하지 못하고 서구 중심적 표현으로 치환하는 경향 확인
